# Contributing to Bolo

感谢你对 Bolo Hugo 主题的关注和贡献！

## 如何贡献

### 报告 Bug

如果你发现了 Bug，请创建一个 Issue 并包含以下信息：

1. 简洁描述性的标题
2. 详细的 Bug 描述
3. 复现步骤
4. 预期行为
5. 实际行为
6. 截图（如果适用）
7. 环境信息（Hugo 版本、操作系统、浏览器等）

### 提出新功能

如果你想提出新功能建议：

1. 先搜索现有的 Issues，看看是否已经有类似的建议
2. 如果没有，创建一个新的 Issue
3. 清晰描述你想要的功能和原因

### 提交代码

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交你的更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建一个 Pull Request

### 代码规范

- 保持代码风格一致
- HTML 模板使用 2 个空格缩进
- CSS/JS 保持现有代码风格
- 添加必要的注释说明复杂逻辑

### 开发环境

```bash
# 克隆仓库
git clone https://github.com/yourname/bolo.git

# 进入示例站点目录
cd bolo/exampleSite

# 启动 Hugo 开发服务器
hugo server --themesDir ../..
```

## 许可证

提交代码即表示你同意将你的代码以 MIT 许可证发布。

## 行为准则

- 友善和尊重他人
- 接受建设性批评
- 关注对社区最有利的事情

感谢你的贡献！
